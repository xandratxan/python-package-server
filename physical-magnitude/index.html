physical-magnitude

Simple operations with magnitudes including units and uncertainties.

WARNING: This package is under active development. The current version is considered non-stable.

Description: This package allows to perform simple operations with magnitudes including units and uncertainties. It allows to define magnitudes with value, uncertainty and unit. It allows to compute simple operations providing the result not only for the magnitude value, but also its uncertainty and unit. Available operations are summation, subtraction, multiplication and division.

Dependency specification: Include the entry dependencies = ['physical-magnitude'] in pyproject.toml
Installation: Via pip:
pip install physical-magnitude --extra-index-url https://xandratxan.github.io/python-package-server/
